# next-template

## Setup

```bash
$ git clone https://github.com/rn1k/nextjs-template.git
$ cd nextjs-template
$ yarn install
```

## Development

### ローカルで実行する場合

```bash
$ yarn dev
$ open https://localhost:3000
```

## Test

```bash
$ yarn test
```

## Storybook

```bash
$ yarn storybook
```
